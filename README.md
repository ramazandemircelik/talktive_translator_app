## ğŸ¯ Usage

### Basic Translation

1. Select source and target languages from the dropdowns
2. Tap the microphone button
3. Speak clearly into your device
4. Watch as your speech is transcribed and translated in real-time
5. Listen to the translated audio

### Two-Way Conversation

1. User 1 speaks in their language
2. App translates to User 2's language
3. User 2 responds in their language
4. App translates back to User 1's language
5. Conversation continues seamlessly

### Settings

- **Speaker Switch**: Manually switch between User 1 and User 2
